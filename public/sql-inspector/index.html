SQL Inspector : Clojure + JDBC + JavaFX Desktop application

This mini application is called “SQL Inspector”.

It is a desktop JavaFX application that connects to a database via JDBC. All tables of that database are shown. You can apply a filter if you wish. Selecting a table will show all its columns and types.

I’ll use the clojure libraries cljfx and next-jdbc

Prerequisites

You need to download JavaFX from Gluon. At the moment of writen version 17.0.2 is the version with Long Term Suppport.

You also need a database. I use a Docker container with Microsoft SQL Server, because that is the database I’m currently use at work.
