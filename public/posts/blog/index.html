SQL Inspector : Clojure + JDBC + JavaFX Desktop application

What?

“SQL Inspector” is a desktop application written in Clojure. It shows all tables of a database. There is a textbox that you can use to filter the tables. Selecting a table will show all its columns and types.

Technology

I’ll use the clojure libraries cljfx and next-jdbc

Step 01 - Database connection

You need to download JavaFX from Gluon. At the moment of writing, version 17.0.2 is the most recent version with Long Term Support.

You also need a database. I use a Docker container with Microsoft SQL Server, because that is the database I’m currently use at work.
